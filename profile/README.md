# about the project
- This project is an expansion of the base game Friday Night Funkin' that uses the playable character Pico.
Its programming is done in the Psych Engine. The mod focuses on the character Pico and his group, such as Darnell, Nene, and Cassandra, among others. It will also focus on his boyfriend, as there are rap battles where he fights against Pico. I think that's all.
